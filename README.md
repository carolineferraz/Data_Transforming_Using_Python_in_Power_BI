<h1>Transformação de Dados utilizando Python no Power BI</h1>

<p>Atividade feita durante o curso de Engenharia de Dados da Gama Academy, onde foi pedido que fosse feita a importação de um arquivo csv para o Power BI via script Python e que nesse mesmo script fosse realizada a transformação dos dados da tabela. Após a importação da base de dados transformada, foi solicitado que fossem adicionados gráficos de barras ou pizza para apresentação dos dados no Power BI, e em seguida, que o script python também fosse rodado em uma máquina virtual Linux da nuvem da Azure.</p>
